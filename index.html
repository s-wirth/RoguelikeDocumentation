Media Project II

Roguelike - Game

Base

The game is a very traditional Roguelike. It consists of a levelsystem, through which the player, has to fight their way to the top level. Each level has 4-5 sublevels. I.e. you have to get through the "Sewer Level", which consists of 5 levels itself, connected whith doors, stairs, etc.
You will choose a character at the beginning of the game, who you will level up through fighting and killing enemies and using items. For simplicity there will be only one character in the beginning.
Similiar games are NetHack, PixelDungeon, etc.
To begin with the game will include only one level, one hero type and the basic enemy types, to create a fully playable game with a proper code structure.

Anti-Hero, Protagonist

The protagonist starts off with a basic dagger(3dm), no armor(2dmc), an amulet with the 5% possibility to block any damage and 50 points health.
Every 2 turns 1 healthpoint is regained.
One step, hit, wait equals one turn.

Enemies

Mutant Rat

Rats that are abnormally big and agressive, due to the magical waste flushed down the sewer.
Mutant Rats start off with 20 points health, no weapon(2dm) and no armor(1dmc).
Drop: Low amount of gold(s), Meat(o)

Insane Outcast

Poor souls that were abandoned by the villagers, because of their mental state.
Insane Outcasts start with 30 points health, basic cloak
Drop: Insane Notes(s), Basic Cloak(s), Amulets(an)

Boss Fight

Villagers don't like the inhabitants of the sewer up in the city. So the pest control tries and make sure that what lives down there, stays down there.

Pest Control

The Pest Control is a huge man in armor with a morning star.
Health: 70 points DM: average 7 points DMC: average 5 points

  • Legend:

  • dm - damage
  • dmc - damage control
  • an - almost never (5%)
  • s - sometimes (15%)
